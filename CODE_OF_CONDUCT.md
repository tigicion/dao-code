# 贡献者行为准则 / Code of Conduct

## 我们的承诺

为营造开放、友好的环境,我们承诺让参与本项目成为不受骚扰的体验,无论年龄、体型、可见或不可见的残障、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 我们的准则

促进积极环境的行为包括:表达善意与同理、尊重不同观点、得体地给予与接受反馈、为错误负责并致歉、以社区整体利益为先。

不可接受的行为包括:性化的言语或图像及任何形式的性骚扰;挑衅、侮辱或贬损性评论及人身/政治攻击;公开或私下骚扰;未经明确许可公开他人隐私信息;在专业场合不当的其他行为。

## 执行

维护者负责澄清与执行准则,可移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 等,并在适当时说明理由。

## 上报

如遇滥用、骚扰或其他不可接受行为,请通过仓库 **Security → Report a vulnerability**(私密渠道)或直接联系维护者 **tigicion** 上报。所有投诉都会被及时、公正地审阅与处理,并尊重举报者隐私与安全。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
