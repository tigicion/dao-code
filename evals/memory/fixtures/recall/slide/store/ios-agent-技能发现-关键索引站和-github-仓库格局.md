**排行榜/目录站**：① skillleaderboard.com——按 GitHub star 日更排行榜，可筛选 Gaming/iOS 分类（此次发现 communityauthor4/Claude-Code-Game-Studios ⭐22,440 即通过此站）；② skillsmp.com——1.8M 技能搜索引擎，按 creator/keyword 筛选；③ communityauthor1/Swift-Agent-Skills——社区维护者 维护的 Swift 技能精选索引（人工审核，质量最高）。
